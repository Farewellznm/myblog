下载命令

从集群下载

scp remote_username@remote_ip:remote_folder  ./ 拷贝到当前目录

免密登录

scp免密码拷贝和ssh免密码登录（https://www.cnblogs.com/liuyihua1992/p/9602324.html） 如果当前用户目录下没有.ssh 文件，原因是没有登录过root，不会记录密码信息

ssh localhost

断点续传

–append 断点续传

rsync -P --rsh=ssh --append  sc03@10.12.17.2:dir  ./

自动运行

可以将命令的输出结果输出到标准输出设备，多用于周期性执行命令/定时执行命令； （https://www.cnblogs.com/peida/archive/2012/12/31/2840241.html)

watch -n 20 command

数据校验

空洞文件https://support.huaweicloud.com/intl/zh-cn/sms_faq/sms_faq_0039.html 通过获取文件的 MD5/SHA1/SHA256 等校验值对比文件的完整性、一致性，可以用来判断文件是否重复、内容是否被修改过等，比较常用的哈希算法有 MD5 与 SHA1（https://cloud.tencent.com/developer/article/1642336）

md5sum file > md5
sha256sum file > sha256
